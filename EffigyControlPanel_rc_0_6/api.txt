API cmd	- Function Description

0 	- Select Pedal Preset
1 	- Set single value in a preset
2 	- Send preset to pedal
3 	- Get preset from pedal
4 	- Get slot from pedal
5 	- Send slot to pedal
6 	- Set mode in current preset
7 	- User Mode Switch 
8 	- Set sensitivity % (knob)
9 	- Recalibrate sensors
10/0xA 	- Recalibrate w/#samples
11/0xB 	- Reboot immediately
12/0xC 	- Change all live channels
13/0xD 	- Set Porch size
14/0xE 	- Set Boot mode
15/0xF 	- Set Ceiling size 
16/0x10 - Set LED max brightness %
17/0x11 - Get system Block
18/0x12 - Get bank from pedal
19/0x13 - Set preset to load
20/0x14 - Get pedal software version
